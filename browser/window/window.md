# [Window](https://developer.mozilla.org/en-US/docs/Web/API/Window)

自动实例化对象window，属性方法不需要通过window调用

## Instance properties

`document` Document对象

`location` url对象

## Instance methods

atob() base64解码

alert() Displays an alert dialog.

btoa() base64编码

setInterval() 每多少毫秒执行函数

setTimeout() 多少毫秒后执行函数
